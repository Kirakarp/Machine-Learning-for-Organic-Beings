What is Machine Learning?

Machine learning is the ability for computers to process data and predict output! Basically, that means that if I show the computer a million pictures of cats, it should be able to guess which of these pictures is a cat!

How does a computer make a prediction?

Using the power of math and statistics! Ever heard of trend lines and linear regression? See below for more!

What sort of data will help my ML model make more accurate predictions?

Good data can be hard to get. Luckily, there are many online websites that allow you to download datasets for free! (A dataset is something that contains pure information, for instance, like the model, weight and CO2 emissions of a collection of cars).

What's linear regression?

Linear regression is basically something we use in statistics to predict the output of an unknown value. What we do is take all the data and draw a line that kinda looks like it averages out where each output point is. However, as humans its hard for us to draw a line through the data and say that what we drew is the perfect average. That's where computers and machine learning comes in!
